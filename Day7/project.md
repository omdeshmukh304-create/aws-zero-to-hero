# 🚀 Day 7 – AWS CI/CD Pipeline with CodePipeline, CodeBuild & CodeDeploy

## 🎯 Objective

Built a complete CI/CD pipeline for a Node.js Todo application using AWS services. The pipeline automatically builds and deploys the application to an Amazon EC2 instance whenever code is pushed to the GitHub repository.

---

# 🏗️ Architecture

```
Developer
    │
git push
    │
    ▼
GitHub Repository
    │
    ▼
AWS CodePipeline
    │
    ▼
AWS CodeBuild
    │
    ▼
Amazon S3 (Artifacts)
    │
    ▼
AWS CodeDeploy
    │
    ▼
Amazon EC2
```

---
