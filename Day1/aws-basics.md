#  What is Cloud Computing?

Cloud Computing is the delivery of computing resources such as **servers, storage, databases, networking, and software over the internet** instead of owning and maintaining physical hardware.

### Key Points

- Access computing resources on demand.
- Pay only for the resources you use.
- Easily scale resources up or down.
- Eliminate the need for physical infrastructure.

---

#  What is AWS?

**Amazon Web Services (AWS)** is Amazon's cloud computing platform that provides **200+ cloud services** for building, deploying, and managing applications.

AWS helps developers and organizations by providing reliable, scalable, and cost-effective cloud infrastructure.

---

#  AWS Pricing Models

AWS offers multiple pricing options depending on business requirements.

## 1️ Pay-as-you-go

With this pricing model, I pay only for the AWS resources that I actually use.

### Best For

- Learning AWS
- Testing environments
- Startups
- Short-term projects

### Advantages

- No upfront cost
- Flexible
- No long-term commitment

---

## 2️ Reserved Instances

Reserved Instances allow users to reserve AWS resources for **1 or 3 years** in exchange for significant discounts.

### Best For

- Production workloads
- Long-term applications
- Predictable usage

### Advantages

- Lower cost
- Long-term savings
- Stable pricing

---

## 3️ Spot Instances

Spot Instances use AWS's unused EC2 capacity and are available at heavily discounted prices.

### Best For

- Batch processing
- AI/ML workloads
- Testing
- Big data processing

### Advantages

- Lowest pricing option
- Can save up to 90%

### Limitation

AWS can terminate Spot Instances at any time if the capacity is required.

#  Cloud Computing Models

Cloud computing services are mainly divided into three categories.

## Infrastructure as a Service (IaaS)

In the IaaS model, AWS provides virtual infrastructure such as servers, storage, and networking, while I manage the operating system and applications.

### Example

- Amazon EC2

---

## Platform as a Service (PaaS)

In the PaaS model, AWS manages the infrastructure and operating system, allowing developers to focus only on writing and deploying code.

### Example

- AWS Elastic Beanstalk

---

## Software as a Service (SaaS)

SaaS provides ready-to-use software over the internet without requiring users to install or manage infrastructure.

### Examples

- Gmail
- Microsoft 365
- Slack

---

#  Core AWS Services

## Amazon EC2

Amazon EC2 provides virtual servers in the cloud.

### Common Use Cases

- Hosting websites
- Running applications
- Deploying Docker containers
- Development and testing

---

## Amazon S3

Amazon S3 is an object storage service used to store and retrieve data securely.

### Common Use Cases

- File storage
- Image storage
- Video storage
- Backups
- Static website hosting

---

## AWS Lambda

AWS Lambda is a serverless computing service that allows code execution without managing servers.

### Features

- Event-driven execution
- Automatic scaling
- Pay only for execution time
